# GDS.FM

A minimal macOS menubar app for streaming [GDS.FM](https://gds.fm) radio.

## Features

- **Left-click** to play/pause
- **Right-click** to open menu with current track info
- Autoplay on launch
- Shows current show name and track (artist — title)

## Requirements

- macOS 13.0+
- Xcode 15+

## Build

1. Open `GDS.FM.xcodeproj` in Xcode
2. Build and run (⌘R)

For release builds, notarization, and CI/CD setup, see [docs/BUILD.md](docs/BUILD.md).
